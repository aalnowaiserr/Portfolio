01 About

Security-minded from the first commit

I'm a Computer Science graduate from the University of Jeddah (GPA 4.40/5.00, Second Honors) who treats security as a design constraint, not a checklist item you tack on before release.

At the National Center of Meteorology I worked the other side of the problem — system hardening, endpoint configuration, software deployment and network access control on live organizational infrastructure. That experience changed how I build: I now think about threat surface, least privilege and data integrity while the architecture is still on the whiteboard.

The result is applications that hold up in the real world — an AI radiology desktop tool that keeps medical data protected end to end, and a bilingual booking platform where every read and write is gated by explicit access rules.

  • Secure by default Hardened endpoints, least-privilege access, encrypted storage and transport.
  • Systems & networking Linux, network architecture, access control and monitoring in production environments.
  • Cloud & delivery AWS services, Firebase security rules, and full-stack delivery from model to UI.
