# Keep the PDFs (which carry personal contact details) out of search results.
#
# NOTE — this is a deliberate downgrade, not an oversight.
#
# The stronger control is `X-Robots-Tag: noindex`, served as a response header:
# it lets a crawler fetch the file, read the header, and drop it from the index
# entirely. A `Disallow` is weaker — it only blocks the *fetch*, so Google can
# still list the bare URL if another site links to it.
#
# We use the weaker one because GitHub Pages cannot send custom response headers
# at all. There is no `_headers`, no `.htaccess`, no config. `X-Robots-Tag` is
# simply not achievable here, so `Disallow` is the only lever that exists.
#
# If this site ever moves behind Cloudflare or any host with header control:
# delete these two lines and serve `X-Robots-Tag: noindex` instead. Do not do
# both — a Disallow would stop the crawler ever seeing the noindex.
Disallow: /AbdulrahmansCV.pdf
Disallow: /agile-scrum-master-certificate.pdf
